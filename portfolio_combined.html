제가 일하는 방식
📚
언어와 기술에 얽매이지 않고 결과로 증명합니다
Python, Java, C#, TypeScript 등 프로젝트 환경에 맞춰 기술을 빠르게 습득합니다. 특정 스택에 갇히지 않고 비즈니스 문제 해결에 집중하며, 새로운 기술도 실무 투입 수준까지 빠르게 끌어올립니다.
🤖
AI를 단순 도구가 아닌 '개발 파트너'로 활용합니다
Claude Code와 Google Antigravity를 실무에 적극 도입하고, 아키텍처 설계부터 QA까지 전 주기에 맞는 커스텀 Agent Skill을 직접 제작합니다. 단순 활용을 넘어, MCP Protocol 기반 도구 서버를 개발해 AI가 더 잘 동작하도록 돕는 인프라 설계까지 관심 영역을 넓히고 있습니다.
🧪
"완벽한 코드는 없다"는 신념으로 품질을 높입니다
JUnit(Java)과 C# 테스트 프로젝트(xUnit/MSTest)를 활용하여 테스트 코드 작성을 생활화합니다. 인터페이스 기반 추상화와 다형성을 적극 도입해 변경에 유연한 구조를 설계하며, 비즈니스 로직과 구현체를 분리하여 결합도를 낮춘 클린 코드를 지향합니다.
💬
모호함 속에서 소통으로 명확한 설계를 이끌어냅니다
모호한 상태에서 코드를 짜기보다 적극적인 소통으로 설계 의도를 먼저 파악합니다. 맡은 일은 끝까지 해내는 책임감을 바탕으로, 동료가 안심하고 협업할 수 있는 든든한 파트너를 지향합니다.
